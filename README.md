# WWfinal

Repositório contendo o projeto WiseWallet integrado com Gemini.

Estrutura principal:
- `wisewallet-app/` – aplicação front-end e scripts de servidor
- `supabase/` – funções e configuração do Supabase

Instruções rápidas:
1. Instalar dependências dentro de `wisewallet-app/wisewallet-app` com `npm install`.
2. Configurar chaves de API conforme `wisewallet-app/README.md` e `wisewallet-app/API_KEYS_SETUP.md`.
